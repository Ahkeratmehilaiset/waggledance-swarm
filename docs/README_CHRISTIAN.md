# Dokumentit Christianille

## Luodut tiedostot:

1. **email_christian.md** - Valmis sähköpostiviesti
2. **latency_comparison.html** - Latenssivertailu (avaa selaimessa → tulosta PDF:ksi)
3. **latency_comparison.md** - Sama markdown-muodossa
4. **latency_comparison.txt** - Sama tekstitiedostona

## Kuinka käyttää:

### 1. Sähköposti:
```
1. Avaa: docs/email_christian.md
2. Kopioi sisältö sähköpostiohjelmaan
3. Lisää nimesi loppuun
4. Liitä mukaan: latency_comparison.pdf (katso kohta 2)
5. Lähetä!
```

### 2. PDF:n luonti:
```
1. Avaa selaimessa: docs/latency_comparison.html
2. Paina Ctrl+P (Print)
3. Valitse "Save as PDF" tai "Tallenna PDF:ksi"
4. Tallenna nimellä: waggledance_latency_comparison.pdf
5. Liitä sähköpostiin
```

### 3. Vaihtoehto (jos selain ei toimi):
```
Lähetä latency_comparison.md tai .txt tekstitiedostona.
Kaikki tärkeä data on siinä.
```
